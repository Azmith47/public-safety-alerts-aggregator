# Data Collection

This folder contains scripts and services responsible for collecting public safety alerts from external sources.

Data sources may include:

- NSW Rural Fire Service (RFS)
- Bureau of Meteorology (BoM)
- NSW State Emergency Service (SES)
- Road incident feeds

Collection methods:

- RSS feeds
- Public APIs
- Permitted web scraping

Collected data will be processed, normalised, and stored in the MySQL database.
